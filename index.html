Sobre Mí

Arquitecto de Software y Visionario Técnico

Soy un Arquitecto de Software con un enfoque de 'Artesano': mi especialidad es el desarrollo de soluciones full-stack [cite: 56] y mi obsesión es entregar productos 'pulidos' con una alta exigencia estética y funcional[cite: 53]. Mi filosofía de trabajo es la 'Libertad Estructurada'[cite: 43, 103]: ofrezco liderazgo técnico y autonomía para construir la mejor solución, dentro de un marco de propósito y revisiones claras. No solo entrego código; entrego un producto robusto y bien pensado.

Me dedico a crear herramientas accesibles y a posicionar proyectos que generen un impacto social. Creo firmemente en la filosofía de "compartir lo aprendido para que otros avancen más rápido"[cite: 35].

El Objetivo: Diseñar una plataforma de agricultura de precisión (v3.0) que funcione como un "gemelo digital" de un sembradío. El sistema evolucionó desde un simple detector de enfermedades en hojas de aguacate a una plataforma integral que combina análisis de IA con monitoreo de datos ambientales simulados.

Características Clave:

El Objetivo: Crear una aplicación web (v5.5) para el análisis microscópico asistido en tiempo real, diseñada como una herramienta potente y accesible para entornos con recursos limitados.

Características Clave:

El Objetivo: Desarrollar una aplicación web (v1.2) para la gestión eficiente de inventarios que asegura la rotación de productos usando el principio FIFO (First-In, First-Out). El sistema digitaliza el registro, seguimiento y despacho de lotes para eliminar la ambigüedad.

Características Clave:

  • Lógica de negocio central que identifica y muestra automáticamente el próximo lote a despachar.
  • Gestión de ubicaciones físicas del almacén (ej. A1, B2), controlando su estado (`empty` u `occupied`) para evitar errores de asignación.
  • Dashboard en tiempo real con KPIs, gráficos de inventario (Chart.js) y un mapa visual del almacén que señala la ubicación exacta del próximo lote a retirar.
FastAPI Python JavaScript Chart.js jsPDF MySQL

El Objetivo: Construir un sistema robusto (v2.0) para la gestión integral de reportes de laboratorio. La aplicación maneja el ciclo CRUD (Crear, Leer, Editar, Actualizar, Eliminar) completo para 22 tipos de estudios diferentes, mapeados a través de 9 tablas de resultados.

Características Clave:

  • Implementación de cálculos automáticos en tiempo real en el frontend (ej. *globulina = proteínas_totales - albumina*) usando la función `watch` de Vue.js.
  • Arquitectura de datos compleja con lógica de mapeo para guardar y actualizar 22 estudios en 9 tablas relacionadas usando `updateOrCreate` de Eloquent dentro de una transacción segura.
  • Depuración avanzada de todo el flujo de datos, resolviendo `RelationNotFoundException` en el backend, errores de tipos de datos (string "N/A" en columnas `double`) y estandarización de claves.
Laravel PHP Vue.js Inertia.js MySQL TailwindCSS
