Sobre Mí

Arquitecto de Soluciones y Líder Técnico

Soy un Arquitecto de Software con un enfoque de 'Artesano': mi especialidad es el desarrollo de soluciones full-stack y mi obsesión es entregar productos 'pulidos' con una alta exigencia estética y funcional.

Mi filosofía de trabajo es la 'Libertad Estructurada': ofrezco liderazgo técnico y autonomía para construir la mejor solución, dentro de un marco de propósito y revisiones claras. No solo entrego código; entrego un producto robusto y bien pensado.

Proyectos Destacados

Un vistazo a la arquitectura, ejecución y resolución de problemas en proyectos reales.

Validación: Proyecto seleccionado para competir en la Feria Nacional de Ciencias Mexicanas (FEMECI) en marzo de 2026, validando la arquitectura técnica y el modelo de negocio.

El Problema de Negocio: La industria del aguacate sufre pérdidas de más del 30% por detección tardía de plagas y mala gestión del suelo.

Mi Solución (Arquitectura Asíncrona): Desarrollé una plataforma con FastAPI que actúa como centro de mando proactivo. Conecta el síntoma (detectado por visión computacional con YOLOv8) con la causa (datos de sensores IoT).

El Valor (Resultado): Entregamos al agricultor métricas precisas para la toma de decisiones, optimizando procesos para aumentar el rendimiento por hectárea.

FastAPI Python YOLOv8n JavaScript Leaflet.js PostgreSQL

El Problema de Negocio: Mantener bases de código separadas para aplicaciones de escritorio (Windows/Mac) y móviles (Android) duplica los tiempos y costos de desarrollo, además de fragmentar la interfaz.

Mi Solución (Desarrollo Multiplataforma): Diseño y desarrollo de una aplicación construida sobre un framework basado en Chromium (arquitectura similar a Discord). Esto me permite compartir el núcleo lógico y la UI, empaquetándola de forma nativa para ambos entornos.

El Valor (Escalabilidad): Despliegue ágil de nuevas características en ecosistemas de escritorio y Android simultáneamente, garantizando una experiencia de usuario estandarizada y reduciendo radicalmente el tiempo de mantenimiento.

JavaScript TypeScript Chromium App Android App Escritorio

El Reto Empresarial: Modernizar y automatizar la captura de más de 22 tipos de estudios clínicos, garantizando la trazabilidad, cálculos precisos en tiempo real y la integridad de los datos médicos.

Mi Solución (Arquitectura B2B): Dirigí el desarrollo de la plataforma y el diseño del esquema de base de datos relacional. Actualmente lidero el proyecto en colaboración con ingenieros de Adaca Medical para integrar cálculos bioquímicos dinámicos directamente en la interfaz.

El Valor (Validación Comercial): El rigor técnico de la plataforma logró asegurar una inversión inicial corporativa, demostrando mi capacidad para ejecutar arquitecturas de grado empresarial.

Laravel Vue.js MySQL / PostgreSQL TailwindCSS Arquitectura B2B

El Problema de Negocio: El análisis microscópico es un proceso lento y costoso que depende 100% de un experto en sitio, creando un "cuello de botella" operativo.

Mi Solución (Optimización de Procesos): Una plataforma web que democratiza el análisis experto. Utiliza IA (YOLOv8) para la detección automática y un asistente (GPT-4) para guiar al personal no-experto.

El Valor (Reducción de Costos): Reduce el tiempo de análisis de minutos a segundos y permite que analistas junior realicen tareas de un senior, optimizando radicalmente el flujo de trabajo del laboratorio.

FastAPI Python YOLOv8 GPT-4 API JavaScript TailwindCSS

El Problema de Negocio: Una mala rotación de inventario genera enormes mermas por productos caducados y eleva los costos de almacenamiento.

Mi Solución (Optimización de Procesos): Un sistema que garantiza digitalmente la rotación FIFO. La plataforma identifica automáticamente el lote correcto y usa un mapa visual del almacén para agilizar el despacho.

El Valor (Reducción de Costos): Elimina las mermas por caducidad y reduce los errores humanos, optimizando toda la operación logística del almacén.

FastAPI Python JavaScript Lógica FIFO MySQL
